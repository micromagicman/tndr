About extension

This extension for Google Chrome will help you collect the greatest number of mutual likes on tinder.com due to the automatic affixing of likes to every girl or man in your area.

Installation

  1. Download the extension.
  2. Extract the files into their own folder. Give the extension files a permanent home. Chrome will need continual access to these files for the extension to work, so the file path cannot change. This means don’t leave the files on your desktop or downloads folder where you may accidentally tidy them up or delete them later.
  3. In Chrome, navigate to chrome://extensions Now drag and drop the extension from your file manager anywhere onto the extensions page in Chrome.
  4. Done! The extension is installed and you are ready to go. You can now manage the extension as you would any other, and its icon will appear in the Chrome Menu (if it has one) to the right of the address bar.

How to use it?

  1. Go to Tinder matches page
  2. Click on the tndr extension icon
  1. Configure pause time between swipes
  2. Once you're ready, click "play"
